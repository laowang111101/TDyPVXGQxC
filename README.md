# 图书管理系统 java579

## 项目概述

图书管理系统是一款基于前后端分离架构的Web应用程序，采用Vue与Java SpringBoot技术栈开发。该系统旨在为图书馆提供便捷的书籍与读者管理功能。

## 技术栈

- **系统环境**：Windows
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven
  - MySQL 8.0
- **技术栈**：
  - Java
  - MySQL
  - SpringBoot
  - MyBatis
  - Vue

## 功能模块

### 系统角色

系统分为两个主要角色：管理员与普通用户。

#### 管理员

- **读者管理**：对读者信息进行增删改查。
- **书籍管理**：对图书信息进行增删改查。
- **借阅管理**：查看和管理图书借阅情况。
- **书籍类型管理**：管理图书分类。

#### 普通用户

- **借阅图书**：查询图书信息，并借阅图书。
- **查看归还状态**：查看借阅图书的归还情况。

## 运行环境

- 前端：现代浏览器
- 后端：Java运行环境

## 部署步骤

1. 准备Java运行环境，并确保Maven和MySQL服务可用。
2. 克隆或下载项目源码。
3. 使用IDEA等开发工具打开项目，并配置相应的数据库信息。
4. 执行Maven构建命令，打包项目。
5. 部署前端代码至Web服务器。
6. 部署后端代码至Java应用服务器。
